Apa itu WEB ?

WEB merupakan layanan internet yang berjalan pada protokol aplikasi HTTP

Definisi Internet :

Internet meurpakan jaringan yang sangat besar, jaringan besar yang menghubungkan kejaringan kecil menggunakan jaringan komunikasi diseluruh dunia protokol TCP/IP

Perbedaan aplikasi berbasis Internet vs WEB :

Aplikasi berbasis Internet :

Menggunakan protokol aplikasi yang sudah ada atau mendefinisikan protokol sendiri.

Aplikasi disisi server berkomunikasi langsung dengan client. Aplikasi disisi client dapat berupa aplikasi yang berdiri sendiri atau ditempelkan dalam aplikasi lain.

Aplikasi berbasis WEB :

Menggunakan protokol HTTP.

Aplikasi disisi server berkomunikasi langsung dengan cilent melalui web server.

Aplikasi disisi client umumnya berupa web browser.

URL (uniform resource locator) :

Adalah sistem pengalamatan file yang digunakan internet.

Format URL : standar dideskripsikan di RFC 1738 (http://www.ietf.org/rfc/rcf1738.txt).

WEB server :

Merupakan suatu software yang dijalankan pada komputer server. Berfungsi agar dokumen web yang disimpan diserver dapat diakses oleh user internet.

HTML : Hyper Text Markup Language

Standar bahasa yang digunakan untuk menampilkan dokumen web, berfungsi sebagai:

Mengontrol tampilan konten

Memuat form untuk transaksi data

Memuat obyek lain seperti image, audio, video, dll

Browser :

Software di client untuk menerjemahkan tag HTML halaman web IE, Chrome, Firefox, Opera, dll

Editor :

Program untuk menulis tag HTML

Notepad, Notepad++, Sublime, Dreamweaver, dll

Struktur Dokumen HTML :

Document HTML bisa di bagi mejadi tiga bagian utama: html, head, dan body.

Setiap document HTML harus di awali dan di tutup dengan tag

Pemformatan Tabel yaitu :

Perbedaan GET dan PSOT :

GET :

Metode default

Data terlihat di alamat browser

Panjang URL terbatas 3000 karakter

Berguna untuk bookmark

Untuk data yang tidak sensitif

POST :

Harus didefinisikan

Data tidak terlihat di alamat browser

Tidak ada limitasi

Tidak bisa di-bookmark

Untuk pengiriman data sensitive. Misal: data login

Select :

Untuk menampilkan pilihan dalam bentuk dropdown Secara DEFAULT, opsi pertama yang akan terpilih Untuk mendefinisikan opsi yang terpilih selain DEFAULT, atribut selected dapat ditambahkan

Atribut Multiple :

Untuk memilih lebih dari satu pilihan menggunakan atribut

Textarea :

rows menunjukkan jumlah baris yang terlihat

cols menunjukkan lebar area yang terlihat

Button : Mendefinisikan tombol yang bisa di-klik

Input : adalah elemen paling penting dalam form input dapat ditampilkan dalam beberapa cara bergantung pada typenya

Atribut Input :

value : nilai awal dari sebuah input

readonly : hanya dapat dibaca (tidak dapat dirubah nilainya)

disabled : tidak dapat digunakan dan di-klik. Nilainya tidak akan dikirim ketika submit form

size : ukuran (dalam jumlah karakter)

maxlength : ukuran maksimal yang diijinkan dalam sebuah input

CSS : Cascading Style Sheet

Berfungsi mendefenisikan style untuk suatu teks dengan jenis huruf, ukuran , warna tertentu.

Metode Pembuatan :

Style Sheet Inline : Mengetikkan langsung dalam tag html sebagai atribut

Style Sheet Internal : Menggunakan tag style di dalam tag head.

Style Sheet External : Menyimpan informasi style ke dalam sebuah file dengan ekstensi/type file .css dan memanggil file css dalam html dengan tag link yang diletakkan dalam tag head.

Style definitions : Definisi style yang ingin dibuat

ID selctor dan Class :

ID selector didefinisikan sendiri (tunggal), ID selector diawali tanda # ( octothorpe)

CLASS diawali dengan tanda titik ( . )
